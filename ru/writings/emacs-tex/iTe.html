Баннер для фонда поддержки ФП

iTe — это режим, обеспечивающий интерактивное создание и редактирование документов TeX и LaTeX. При работе с ним, можно выбрать отдельные фрагменты, такие как формулы, картинки, подписи и т.п., и затем перемещать, вращать и изменять их размеры. Сам пакет не обеспечивает предварительный просмотр создаваемого документа, но делает работу более удобной.

Домашнюю страницу данного пакета можно найти на сайте CTAN. Она содержит дистрибутив данного пакета, подробное руководство по установке и использованию пакета, а также примеры получаемых документов.

Установка пакета

Для установки пакета сначала скачайте его с домашней страницы проекта, распакуйте. В составе пакета идет набор стилевых файлов LaTeX, которые должны быть доступны при компиляции исходных текстов. Для подключения пакета поместите в файл инициализации Emacs следующие команды:

(add-to-list 'load-path "path_to_ite")
(autoload 'ite "ite"
   "Start the interactive TeX editor if invoked on a (La)TeX file.
   Otherwise, display help." t)
(setq ite-config-files (list "path_to_ite/ite.conf" "~/iTe/ite_local.conf"))

первая команда помещает каталог с iTe в список каталогов для поиска пакетов Emacs, вторая команда позволяет загрузить iTe при первом его вызове, а последняя — задает список файлов конфигурации iTe.

Использование пакета iTe

Для начала работы с пакетом, необходимо лишь выполнить M-x ite, и пакет будет загружен и начнет работать.

Для обеспечения связи между исходным текстом и результирующим файлом используются специальные команды LaTeX1. Для использования данных команд необходимо подключить стилевой файл ite.sty (с помощью команды \usepackage{ite}).

Активные объекты (те, с которыми можно проводить манипуляции) определяются командой \ite, которая вначале задается без параметров, а после визуального редактирования, содержит параметры расположения, размера и наклона. Список управляющих клавиш, с помощью которых производится работа с объектами, приведен в файле bindings.html дистрибутива iTe.

Действия с активными объектами проводятся в рамках специального окружения iteblock, которое задает размеры области в которой будут проводиться действия. Если вы не хотите задавать размеры области, то iTe это может сделать за вас, но для этого необходимо подключить стилевой файл trig.sty
