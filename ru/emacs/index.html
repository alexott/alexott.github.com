Код для Emacs

На этом сайте доступно некоторое количество кода, написанного мной для Emacs:

  • hs-lint.el — модуль поддержки HLint в Emacs (этот пакет также включен в состав HLint). Позволяет получить список замечаний и осуществляет навигацию по этому списку. Используется следующим образом (для запуска команды hs-lint используется сочетание клавиш C-c l):
(require 'hs-lint)
(defun my-haskell-mode-hook ()
   (local-set-key "\C-cl" 'hs-lint))
(add-hook 'haskell-mode-hook 'my-haskell-mode-hook)
  • Исправленная версия blorg.el — исправлено несколько ошибок, в основном в части оформления текста

Небольшие советы

Данные советы по большей части применимы для GNU Emacs версии 21.x и старше:

  • При использовании desktop.el для сохранения списка файлов и переменных при выходе изEmacs может быть полезен такой код, который к сохраняемым данным добавляет еще и кодировку с которой использовался буфер.
(add-to-list 'desktop-locals-to-save 'buffer-file-coding-system)
  • Emacs позволяет указывать нужную кодировку для определенных типов файлов. Это делается добавлением записей в переменную file-coding-system-alist. Записи представляют собой пары значений регулярное выражение/кодировка. Эту переменную можно установить с помощью команды customize-variable.
  • Отправка почты на сервера требующие SMTP Auth делается согласно инструкциям на этой странице.

Большее число различных советов и вариантов настройки вы можете найти в моем блоге, в разделе посвященном Emacs, и в блоге Emacs Tips.
