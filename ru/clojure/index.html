Баннер для фонда поддержки ФП

С некоторых пор я использую язык программирования Clojure для разработки программ, работающих на платформе JVM (я правда не особо люблю язык Java). Clojure хоть и молодой язык, но имеет достаточно большое количество интересных возможностей, таких как неизменяемость данных и упрощенная модель конкуретного программирования, наравне с традиционной функциональностью Lisp'а. А работа на платформе JVM позволяет использовать большое количество библиотек, написанных для данной платформы.

Основные направления использования данного языка в моем случае — анализ данных, как численных, так и текстовых, с целью извлечения разнообразной информации и классификации. Очень полезным инструментом в этом деле является проект Incanter, в разработке которого я участвую.

Источники информации

Для журнала "Практика функционального программирования" я написал вводную статью про язык Clojure, которую можно использовать в качестве введения в программирование на этом языке. Я стараюсь поддерживать ее в актуальном состоянии, отражая изменения между версиями языка. А оригинальную версию, можно найти на сайте журнала. Список имеющейся литературы по Clojure можно найти в обзоре литературы по функциональному программированию.

Ну и конечно, основной источник информации — сайт языка и список рассылки. Полезным для новичков является проект labrepl, который содержит все необходимое для начала работы с языком. И еще один источник информации о Clojure — это Planet Clojure — агрегатор блогов людей, использующих Clojure. Кроме того, имеется достаточно большое количество видео-материалов по языку, ссылки на которые собраны на отдельной странице.

На русский язык также переведена книжка-комикс Casting SPELs with Clojure — русская версия называется Волшебство и магия Clojure.

Из русскоязычных online-ресурсов стоит отметить конференцию clojure на jabber-сервере conference.jabber.ru, а также список рассылки clojure-russian. Кроме этого, существует аггрегатор блогов "Русская планета Clojure".
