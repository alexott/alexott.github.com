Code for Emacs

I wrote some code for Emacs (not counting patches for different projects):

  • hs-lint.el — module for work with HLint from Emacs (it now included into haskell-mode). It allows to get list of hints, and provide navigation through this list. It could be used following way (to run of hs-lint command the C-c l key binding is used):
(require 'hs-lint)
(defun my-haskell-mode-hook ()
   (local-set-key "\C-cl" 'hs-lint))
(add-hook 'haskell-mode-hook 'my-haskell-mode-hook)
  • Patched version of blorg.el — some errors were fixed, mostly in text formatting part
